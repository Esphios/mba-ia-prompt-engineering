# AGENTS.md

## Escopo

Este diretório corresponde ao capítulo sobre enriquecimento de prompts e expansão iterativa.

## Orientações para agentes

- Preserve materiais do capítulo em pt-BR.
- Trate `prompt-enriquecido.md` como a aula principal.
- Use `README.md` para estrutura, execução e importância do capítulo.
- Use `history.md` para divergências antigas, referências removidas e memória editorial.
- Não assuma a existência de `examples/`, `lessons.md` ou `requirements.txt` local sem verificar.
- Não descreva `1-ITER_RETGEN.py` como retrieval externo completo se o script não integrar fontes externas reais.
