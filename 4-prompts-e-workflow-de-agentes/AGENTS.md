# AGENTS.md

## Escopo

Este diretório corresponde a um capítulo specification-driven sobre workflows multiagente.

## Orientações para agentes

- Preserve materiais do capítulo em pt-BR quando estiver editando documentação.
- Trate `prompts-e-workflow-de-agentes.md` como a aula principal.
- Use `README.md` para estrutura, uso e importância do capítulo.
- Use `history.md` para memória editorial e contexto histórico do workflow.
- Não trate a pasta como uma aplicação executável única.
- Preserve a separação entre `commands/` e `agents/`.
- Não invente diretórios, scripts ou fluxos que não estejam especificados no capítulo.
